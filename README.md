# Plezanje.net Graphql API

## Description

Graphql API from plezanje.net website written in [NestJS](https://nestjs.com/).

## Pre reqisites

Node & npm package manager, PostgreSQL server.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
